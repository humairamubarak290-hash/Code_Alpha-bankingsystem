# 🏦 Banking System in C++

A simple menu-driven Banking System developed using C++ and Object-Oriented Programming (OOP) concepts. This project allows users to perform basic banking operations such as deposits, withdrawals, balance checking, and viewing transaction history through an interactive console interface.

--------------------------------------------------

📌 Features

- Deposit Money
- Withdraw Money
- View Account Balance
- Store Transaction History
- Interactive Menu-Driven Interface
- OOP-Based Design

--------------------------------------------------

🧩 Technologies Used

- C++
- Object-Oriented Programming (OOP)
- VS Code
- MinGW Compiler

--------------------------------------------------

📂 Project Structure

├── banking.cpp
└── README.md

🚀 How to Run

Step 1: Open Terminal
Open terminal in VS Code.

Step 2: Navigate to Project Folder

cd "D:\My Codes\Banking system"

Step 3: Compile the Program

g++ banking.cpp -o banking.exe

Step 4: Run the Program

.\banking.exe
